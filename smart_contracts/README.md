# Hardhat Project

## Installation
install [Node.js](https://nodejs.org/en/) before running following command
```shell
npm install hardhat
```

compile contract using
```shell
npx hardhat compile
```
deploy the contract to ropsten test network
```shell
npx hardhat run scripts/deploy.js --network ropsten
```
running the test
```shell
npx hardhat test
```
